# SineBot

SineBot, film ve dizi seçmeyi kolaylaştıran tek sayfalık (single-file) bir web uygulamasıdır. Tür, dil, platform (TR), yıl aralığı ve minimum puan gibi filtrelerle havuzu daraltır; ardından rastgele bir öneri sunar. Arama ekranından içerik arayıp detaylarını da görüntüleyebilirsin.

## Öne çıkanlar
- Keşfet ekranı: filtrelerle rastgele film/dizi önerisi
- Arama: film/dizi adına göre hızlı arama
- Detay sayfası: puanlar, künye, oyuncular, benzer içerikler
- Mobil UX: mobil filtre “bottom sheet” ve sekmeli detay görünümü

## Çalıştırma
Bu proje statik bir `index.html` dosyasıdır. Dosyayı tarayıcıda açman yeterli.
